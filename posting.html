Halo semua, salam kenal nama saya Arif Rahman.

Saya adalah student hacktiv8 batch 39 Majestic fox. Background pendidikan saya adalah Sistem Informasi. Meskipun latar belakang saya IT, tapi saya sadar, ilmu yang saya dapatkan semasa kuliah dulu belum cukup untuk bekal saya berkarir di bidang yang saya jalani, itulah motivasi saya untuk bergabung di hacktiv8

  • What did i get?

Pada hari Senin, tanggal 11 November 2019 itu adalah hari pertama saya sebagai student dari Hacktiv8. Hari pertama diawali dengan berkenalan dengan HTML dan CSS, mungkin bagi yang background IT HTML dan CSS sudah tidak asing lagi, tapi tetap saja kita harus ikuti dan kita dalami kembali. Meskipun background saya dari IT, saya juga belum expert dalam menggunakan HTML ini. Jadi kegiatan nya, kita mulai dengan baca baca refrensi dari web DEN, ngerjain online course dari web Code Academy dan mengerjakan challenge-challenge yang ada di dalam nya, dan hasil dari pembelajaran kita nanti nya akan di screenshoot sebagai bukti progress pembelajaran nya dan di submit kedalam FOX HUB.

Selanjutnya saya juga belajar mengenai CLI, git dan github. Comand line interface atau yang biasa kita panggil CLI adalah tool yang digunakan sebagai pengganti mouse untuk memberikan perintah di laptop kita. Lewat CLI ini kita bisa buat folder baru, file baru, explore folder tanpa perlu menggunakan mouse, pure menggunakan keyboard dan ini adalah ilmu baru untuk saya juga. Meskipun saya kuliah dengan background IT saya tidak pernah atau mungkin lupa jadi saya anggap ini adalah hal baru untuk saya.

Oh iya, untuk belajar di Hacktiv8 kita juga diwajibkan menggunakan linux, dan kenapa kita belajar CLI? karena semua perintah linux kebanyakan menggunakan CLI seperti install software dan upload ke GIT HUB.

Git sendiri merupakan suatu version conrol system yang mengatur berbagai kerjaan kita sedangkan github ini adalah service providernya yang memberikan layanan untuk menggunakan git. Dengan git semua history kerjaan kita akan terekam jadi jika data kita yang terbaru ternyata ada error kita bisa cek ke data sebelumnya yang belum diganti. Apalagi mengingat dalam projek dikerjakan oleh kelompok maka dengan git, pengelolaan file akan lebih teratur dan terdata history perubahannya.

pada akhir minggu, kita akan dibuatkan tim untuk membuat sebuah project web e-commerce beranggotakan 4 orang per kelompok. Tujuan dari pembuatan project ini adalah melatih penggunaan command line serta Git dan Github lebih banyak. Dan setelah project kelompok kita membuat web personal sesuai dengan selera dan kemampuan kita.

Mungkin cukup segitu saja ulasan dari aktifitas saya selama seminggu di Hacktiv8 semoga saja saya tetap bertahan dan melanjutkan phase phase berikutnya
